HTML

Hypertext Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser and defines the content and structure of web content. It's usually the first thing you learn when starting out with webpages.

CSS

CSS, or Cascading Style Sheets, is a style sheet languge used for specifying the styling and presentation ofg a document written in, for example, HTML. This is also the next thing to usually learn after starting with HTML.

JS
